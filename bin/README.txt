This is where third-party binaries are kept.
They're bundled in releases; if you clone this repo you must also get the latest release and copy the binaries therein to this folder.

WARNING: in release 20171023 (v3.0.9) BLAST binaries were updated to ncbi-blast-2.6.0+, you might have to download that release to update your old binaries and remove the ncbi-blast-2.2.27+ folder
